# chapter 5. 텍스트 포매팅
## Intro
보기 좋은 테이블과 모양새를 위해 텍스트의 포맷을 지정하고 싶을 것     
파이썬은 텍스트-문자 서식을 사용하여 출력하는 정보를 수정하거나 개선하는 여러 가지 방법을 제공    

## 5.1 포매팅 연산자
문자열 포매팅 연산자 %는 다음과 같이 사용    
1개 이상의 값 인수가 있다면 해당 인수들은 반드시 튜플 안에 위치해야 함  

```
포맷_문자열 % 값      
포맷_문자열 % (여러_값들)         
```

## 5.2 포맷 지시자
| 지시자 | 의미                                                        |
|:--------|:------------------------------------------------------------|
| `%d` | 10진수 |   
| `%i` | 정수, %d와 동일함 |    
| `%s` | 입력 값의 표준 문자열 표현(모든 데이터 객체의 표준 문자열 표현을 출력) |   
| `%r` | 객체가 파이썬 코드에서 보이는 그대로 출력|    
| `%x` | 16진수   |
| `%X` | A-F 문자가 대문자인 16진수  |     
| `%o` | 8진수    |
| `%u` | 무부호 정수(unsigned integer)    |
| `%f` | 고정소수점 포맷으로 출력하는 부동소수점 숫자     |
| `%F` | %f와 같음    |
| `%e` | 지수 기호를 출력하는 부동소수점 숫자     |
| `%E` | %e와 같지만 대문자 E를 사용함    |
| `%g` | 가장 짧은 기준 표현(canonical representation)을 사용한 부동소수점    |
| `%G` | %g와 같지만 지수 출력 시 대문자 E 사용   |
| `%%` | 리터럴 백분율 기호(%)|


포맷을 사용한 글자의 너비와 소수점 숫자 조정은 다음과 같이 사용   
c는 위 표의 포맷 문자 중 하나를 의미함    
```
%[-][너비][.정밀도]c
```

## 5.3 포매팅 연산자 변수-너비 출력 필드
변수-너비 필드를 만들려면 일반적으로 고정 너비를 명시하는 정수 위치에 별포 기호(*)를 넣음  
포매팅 코드에서 일반적으로 정수를 넣는 곳에 별표 기호를 대신 넣을 수 있으며, 리스트에는 별표 기호와 대응하는 정수가 반드시 존재해야 함       
```
%*c ... %(너비, 출력할 데이터)
%*.*f ... %(앞 너비, 뒷 정밀도, 출력할 데이터)
```

## 5.4 전역 'format' 함수
```
format(데이터, 사양)  
[너비][,][.정밀도][타입_문자]
```

```python
big_n = 10**12
format(big_n, ',')
>>> 1,000,000,000,000
```

| 포맷 명세 | 의미                                                        |
|:--------|:------------------------------------------------------------|
| `','`   | 숫자에 천 단위 위치 구분자를 함께 표기                             |
| `'5'`   | 최소 출력-필드 너비를 5문자로 지시함                              |
| `'10'`  | 최소 출력-필드 너비를 10문자로 설정함                             |
| `'10,'` | 최소 출력-필드 너비를 10문자로 설정하고, 천 단위 위치 구분자도 함께 표기  |
| `'10.5'`| 최소 출력-필드 너비를 10으로 설정하고, 출력할 공간이 부족하면 지수 포맷으로 출력됨|
| `'10,.7'` | 최소 출력-필드 너비를 10으로, 정밀도는 7로 설정, 천 단위 위치 구분자도 함께 표기 |
| `'10.3f'`| 고정소수점 표기. 출력-필드 너비를 10으로 하고 소수점 오른쪽 숫자를 정확히 3개만 표기. 경우에 따라서는 정확한 숫자의 개수를 표기하기 위해 반올림하거나 부족한 자릿수를 0으로 채울 수도 있음|
|`'.3f'`| 소수점 오른쪽 숫자를 정확하게 3개만 출력하며 최소 너비는 설정하지 않음|
|`'b'`| 이진수로 표기|
|`'6b'`| 이진수로 표기. 6개의 문자 필드에 오른쪽 자리 맞춤 수행|
|`'x'`| 16진수로 표기|
|`'6x'`| 16진수로 표기. 6개의 문자 필드에 오른쪽 자리 맞춤 수행|
|`'o'`| 8진수로 표기|
|`'6o'`| 8진수로 표기. 6개의 문자 필드에 오른쪽 자리 맞춤 수행|

## 5.5 format 메서드
포매팅을 완벽하게 제어하려면 format 메서드를 사용   
format 메서드는 전역 format 함수의 모든 기능을 사용하지만 여러 출력 필드를 다룰 수 있어 유연함
```
포맷이_지시된_문자열.format(인수들)
```
```python
print('{} plus {} equals{}.'.format(25, 75, 100))
```

위 표현식은 출력 필드를 제외한 모든 텍스트를 '포맷이_지시된_문자열'을 통해 넘김\
출력 필드는 {}와 대응되며, 각 출력 필드에 '인수들'의 각 항목이 출력됨

데이터 객체를 출력할 때 특별한 포맷을 지정하지 않는다면 단순하게 중괄호 기호 쌍({})을 각 항목에 사용\
문자열은 문자열로 출력되고, 정수는 정수로 출력됨

```python
fss = '{} said, I want {} slices of {}.'

name = 'Pythagoras'
pi = 3.141592
print(fss.format(name, 2, pi))

>>> Pythagoras said, I want 2 slices of 3.141592.
```

중괄호 기호는 특수 문자이며, 중괄호 기호 자체를 출력하고 싶다면 {{와 }}를 사용
```python
print('Set = {{{}, {}}}'.format(1,2))

>>> Set = {1,2}
```

## 5.6 위치로 순서 정하기(이름 혹은 색인)
```
{ [위치] [!r|s|a][:사양]}
```
포맷 문자열의 출력 필드 구문 중 대괄호 기호로 감싼 항목은 필수로 넣을 필요가 없는 선택 사항\
두 번째 항목 안에는 느낌표 기호와 함께 r,s,a 중 하나가 뒤따름\
'사양'은 잠재적으로 복잡한 포매팅 매개변수를 열거함

'위치'는 숫자 혹은 이름으로 어느 인수를 출력할 것인지 가리킴\
'위치'지시자를 사용하면 순서와 상관없이 인수를 지정\
'위치'지시자는 색인 번호나 인수 이름으로 지정할 수 있음

'위치 색인'은 format 메서드의 인수 리스트에서 가리키는 항목의 색인 숫자를 사용하며 색인은 0부터 시작함\
'위치 이름'은 인수 이름과 동일해야 함
```python
print('{2}; {1}; {0}!'.format(10, 20, 30))

>> 30; 20; 10!
```
```python
fss = 'a equals {a}, b equals {b}, c equals {c}.'
print(fss.format(a=10, c=100, b=50))

>>> a equals 10, b equals 50, c equals 100.
```
